# Esquinas • Convite tipo Spotify (lua&thay•fy)

Single-page site inspirado no Spotify, com tema preto + terracota e convite pro show (RSVP + .ics). Agora com **resposta imediata** conforme a seleção (sim/talvez/não).

## Como usar
1. Abra `index.html` no navegador.
2. Edite os dados do convite no topo de `script.js` (data, hora, local, cidade, nota).
3. (Opcional) Substitua a capa em `assets/esquinas-cover.jpg` por uma imagem sua.
4. Publique no GitHub Pages / Vercel se quiser compartilhar com a Luana.

> Projeto pessoal, sem fins comerciais. Não inclui letras/trechos protegidos.


Extras adicionados: galeria de fotos (polaroids), <audio id="bgm">, botão Aceito inicia música, whispers por seção.
Coloque sua trilha em assets/bgm.mp3.
